# OpenWeatherMap API Kurulumu

## Amaç
Tarla bazlı sıcaklık, nem, rüzgar ve yağış verilerini almak.

## Sağlayıcı
OpenWeatherMap

## Durum
OpenWeatherMap hesabı oluşturuldu ve API key alındı. API key aktif hale geldikten sonra İzmir lokasyonu üzerinden test edildi ve başarılı JSON response alındı.

## Örnek Endpoint
```text
https://api.openweathermap.org/data/2.5/weather?lat=38.4622&lon=27.0923&appid=YOUR_API_KEY&units=metric&lang=tr
